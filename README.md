# lc-tool

#### 介绍
lc-tool用于支持指定Java后台框架开发的工具包集合，这些工具包对一些第三方框架再次进行了封装从而能高度适应框架的开发习惯，有助于我们减少编写大量重复的代码，在开发的过程中减少关注一些完全可忽略掉的代码编写，提升开发效率；

#### 项目结构
lc-tool
│
├─lc-mp（基于MyBatis-Plus的本地化适应增强）   
│      
└─待定
